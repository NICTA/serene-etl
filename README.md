## serene-etl

This package is a collection of Python modules for the Serene data integration project. The modules contained here allow flat relational files to be integrated into a combined Solr index.

 * serene_ansible
 * serene_index
 * serene_load
 * serene_metadata
 * serene_platform
 * serene_proxy
 * serene_schema
 * serene_solr_config
